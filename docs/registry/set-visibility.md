# cnb registry set-visibility

```
cnb registry set-visibility <registry> <visibility>
```

设置制品库可见性。

## 参数

| 参数           | 说明                           |
|----------------|--------------------------------|
| `<registry>`   | 制品库路径                     |
| `<visibility>` | 可见性（public/private/secret）|

## 示例

```bash
cnb registry set-visibility my-org/my-registry private
```
