# cnb knowledge

知识库管理，管理仓库的 Embedding 知识库。

## 可用命令

- [cnb knowledge list-models](/knowledge/list-models) — 列出支持的 Embedding 模型
- [cnb knowledge info](/knowledge/info) — 获取知识库信息
- [cnb knowledge clean](/knowledge/clean) — 清除知识库
- [cnb knowledge query](/knowledge/query) — 查询知识库

## 示例

```bash
$ cnb knowledge list-models
$ cnb knowledge info
$ cnb knowledge query
```
