# cnb-rs pull

Pull Request 管理，查看、创建、更新和合并 PR。

## 可用命令

- [cnb-rs pull list](/pull/list) — 列出与我相关的 Pull Request
- [cnb-rs pull create](/pull/create) — 创建 Pull Request
- [cnb-rs pull update](/pull/update) — 更新 Pull Request
- [cnb-rs pull merge](/pull/merge) — 合并 Pull Request

## 示例

```bash
$ cnb-rs pull list
$ cnb-rs pull create
$ cnb-rs pull merge
```
