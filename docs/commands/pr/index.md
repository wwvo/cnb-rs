# cnb-rs pr

PR 管理，查看、创建、更新和合并 PR。

## 可用命令

- [cnb-rs pr list](/pr/list) — 列出与我相关的 Pull Request
- [cnb-rs pr create](/pr/create) — 创建 Pull Request
- [cnb-rs pr update](/pr/update) — 更新 Pull Request
- [cnb-rs pr merge](/pr/merge) — 合并 Pull Request

## 示例

```bash
$ cnb-rs pr list
$ cnb-rs pr create
$ cnb-rs pr merge
```
