# cnb-rs knowledge clean

```
cnb-rs knowledge clean
```

清除当前仓库的知识库。

删除成功后输出 `知识库已清除`。如果知识库不存在，输出对应的提示信息。

## 选项

**继承的全局选项：**

- `--repo <REPO>`: 指定仓库路径（格式：`group/repo`）
- `--domain <DOMAIN>`: 指定目标域名（默认：`cnb.cool`）

## 示例

```bash
$ cnb-rs knowledge clean
知识库已清除
```
