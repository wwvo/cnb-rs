# cnb-rs knowledge list-models

```
cnb-rs knowledge list-models
```

列出平台支持的所有 Embedding 模型。

每行输出一个模型名称。

## 选项

**继承的全局选项：**

- `--domain <DOMAIN>`: 指定目标域名（默认：`cnb.cool`）

## 示例

```bash
$ cnb-rs knowledge list-models
text-embedding-v2
bge-large-zh
```
