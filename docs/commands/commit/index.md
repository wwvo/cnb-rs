# cnb-rs commit

Commit 管理，管理 Commit 附件。

## 可用命令

- [cnb-rs commit asset-stats](/commit/asset-stats) — 统计 Commit 附件大小
- [cnb-rs commit asset-clean](/commit/asset-clean) — 清理 Commit 附件
- [cnb-rs commit asset-upload](/commit/asset-upload) — 上传附件到 Commit

## 示例

```bash
$ cnb-rs commit asset-stats
$ cnb-rs commit asset-upload
```
